# Wes plugin

Plugin to use WES from CartElectronics

# Configuration of the Wes

After installing the plugin, you just need to activate it :

![mobile1](../images/configuration.png)

**Configuration**

To configure the plugin, you must add the Wes.

To add a wes : **Plugins** → **Domotic protocol** → **Wes** → **Add**

![mobile2](../images/ajouter.png)

Here are the parameters to enter :

-   **Name of your WES** : Wes name
-   **Activate** : Wes activation
-   **Wes IP** : inform the WES access ip
-   **Wes File** : you have to put "general" if it's a wes configuration.
-   **Port of Wes** : leave blank if you didn't change this in the configuration of the Wes.
-   **Wes account** : put the login of the Wes account (originally it is "Admin").
-   **Wes password** : put the password of the Wes account (originally it is "wes").

> **Tip**
>
> If you want to change its information you must first go to the WES interface.

![mobile3](../images/wesGlobal.png)

After saving, you will get a multitude of modules, you just need to activate the ones you want.

![mobile4](../images/wesGlobalView.png)

> **Tip**
>
> the grayed modules are not monitored by the plugin, you must activate them for that.

# Relay configuration

After the initialization of the Wes Plugin, you can click on the key see below.

![mobile5](../images/wesGlobalRelais.png)

By clicking on this key a modal is displayed select the relays and button wish and validate. which leads to a scenario on the wes to have real-time feedback of buttons and relays (for other feedback you have to go to the general settings of the plugin.)

![mobile6](../images/wesGlobalView.png)
